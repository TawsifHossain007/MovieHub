# Website Name: MovieHub

**Live Site:** https://musical-kheer-4457de.netlify.app/ 

## Overview
This is a movie collection web application where users can create and manage their personal movie library.

Authenticated users can add movies with details such as title, genre, description, and release year, which are then saved to their My Collection page. Logged-in users can view detailed movie information and add any movie to their Watchlist, allowing them to track movies they plan to watch later.

The application focuses on a smooth user experience with authentication-based access, organized movie management, and personalized collections.

---

## Features

- **Movie Creation**: Users can add movies with details such as title, genre, description, and release year.

- **My Collection**: Each user has a personal collection page displaying all the movies they have added.

- **Watchlist Management**: Users can add or remove movies from their watchlist to keep track of movies they want to watch.

- **Movie Details Page**: Logged-in users can view detailed information for each movie.

- **User Authentication**: Secure login and registration system to ensure user-specific access.

- **Access Control**: Certain features like viewing movie details and managing collections are restricted to authenticated users.

- **Search & Filter Functionality**: Users can easily search and filter movies by title, genre, or release year.

- **Profile-Based Experience**: Movie collections and watchlists are tied to individual user accounts.

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.

- **Secure Backend**: Movie and user data are stored securely with proper authentication and authorization.

- **Clean & Simple UI**: Plain, user-friendly interface focused on easy navigation and usability.

- **Scalable Structure**: Can be extended with features like ratings, reviews, or recommendations in the future. 


---

## Tech Stack

- **Frontend**: React, Tailwind CSS, DaisyUI, Framer Motion
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: Firebase Authentication
